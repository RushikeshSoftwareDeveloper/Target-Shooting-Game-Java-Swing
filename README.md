# 🎯 Target Shooting Game (Java Swing)

## 📌 Project Overview

Target Shooting Game is a simple arcade-style game developed using **Java Swing**.
The player aims using the mouse and shoots a moving target to score points before the timer runs out.

The game demonstrates **event-driven programming, graphics rendering, and game loop logic in Java**.

---

## 🚀 Features

* 🎮 Start Screen
* 🎯 Moving Target
* 🖱 Mouse Click Shooting
* 🎯 Crosshair Cursor
* 📊 Score Tracking System
* ⏱ 30 Second Countdown Timer
* 💀 Game Over Screen
* 🔄 Restart Button

---

## 🛠 Technologies Used

* Java
* Java Swing (JFrame, JPanel)
* Graphics API

## 📊 Game Rules

* Shoot the moving target to score points.
* Each successful hit increases the score.
* The game runs for **30 seconds**.
* After the timer ends → **Game Over**.
